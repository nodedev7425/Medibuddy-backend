## 1. Requirements

- python 3.14
- pipenv

## 2. Getting started

The first step is installing all dependencies in the pipenv virtual environment.
```shell
$ pipenv shell
$ pipenv install
```

Then copy-paste the `.env.example` and change the settings according to your needs.
